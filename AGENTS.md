# Use of AI Agents

During the development of this project, I collaborated with an AI assistant (Gemini) to help structure my capstone. 

* **Where it helped:** The AI helped me conceptualize a project with an appropriate scope for a 5-day timeframe. It also provided the boilerplate syntax for the `argparse` library (which can be tricky to memorize) and helped generate the GitHub Actions YAML file for my CI/CD pipeline.
* **Where it steered me wrong:** At first I could not obtain the green check mark because I was trying to run functions using "python" instead of "python3".
* **What I learned:** I learned how to separate my core Python logic from the CLI interface so that `pytest` can easily test functions without triggering command-line prompts.
